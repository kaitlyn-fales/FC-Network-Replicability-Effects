The contents of this folder are as follows:

**lmm.R:** This is the R script to run all of the linear mixed effect models discussed in the manuscript and supplement. This file also exports the "_regressed" and "_baseline" data files where the random effects are controlled for, and the baseline dataset is constructed for bootstrap, respectively.

**pipeline_atlas_interaction_dmn.R:** This is the R script to examine the pipeline and atlas interaction within the DMN, corresponding to Figure 3 in the manuscript. 

**scan_param_summary.xlsx:** This Excel sheet provides the scanning parameters used to match the scanner brand with the correct data records for the LMMs that include scanner brand in lmm.R.
